# Nishant Kumar - Software Developer Portfolio

## 🚀 About This Project

A stunning glassmorphism portfolio website showcasing the skills, projects, and experience of Nishant Kumar, a Software Developer specializing in ASP.NET MVC, JavaScript, Java, and React.

## 🎨 Design Features

- **Glassmorphism Design**: Modern frosted glass effects with backdrop filters
- **Orbital Skills Section**: Revolutionary 3D orbital layout for technical skills
- **Animated Backgrounds**: Floating particles and dynamic gradients
- **Responsive Design**: Optimized for all devices and screen sizes
- **Smooth Animations**: CSS animations and JavaScript interactions
- **Accessibility**: WCAG 2.1 AA compliant with semantic HTML5

## 📁 Project Structure

## 🚀 Getting Started

1. **Clone or Download**: Get the project files
2. **Open in Browser**: Open `portfolio.html` in your preferred browser
3. **Local Server** (Optional): Use a local server for best results
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   
   # Live Server (VS Code extension)
   Right-click → Open with Live Server
   ```
